Projet 1 - Création du Site Web pour l'Agence Kafonli


Page d'accueil du site. Constitué d'un slider, d'une brève description de l'agence Kafonli, de ses services, ainsi qu'une FAQ pour les visiteurs.

Page "À propos" du site. Description détaillée de l'entreprise et présentation des membres de l'agence Kafonli.

Page "Services" du site. Présentation des différents services que propose l'agence Kafonli, tels que la création de sites web, le design graphique, etc.

Page "Portfolio" du site. Présentation de quelques travaux réalisés par l'agence, afin de mettre en valeur leur expertise.

Page "Contact" du site. Formulaire de contact pour permettre aux utilisateurs de communiquer directement avec l'agence, ainsi que les coordonnées nécessaires.

Page "Rendez-vous" du site. Formulaire pour prendre rendez-vous directement avec l'agence, facilitant les interactions avec les clients.

Page de connexion. Un formulaire de connexion permettant aux administrateurs du site de se connecter et de gérer le contenu du site.

Page "Dashboard". Tableau de bord permettant aux administrateurs de gérer le contenu du site et de suivre les interactions des utilisateurs.
